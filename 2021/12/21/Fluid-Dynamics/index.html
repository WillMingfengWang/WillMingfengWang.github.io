1.1Euler静力学公式

1.1.1推导过程

\[ \begin{aligned} \vec{F_b} &={F_{b,x}}\vec{i}+{F_{b,y}}\vec{j}+{F_{b,z}}\vec{k}\\ &=[{f_{b,x}}\vec{i}+{f_{b,y}}\vec{j}+{f_{b,z}}\vec{k}]\rho dxdydz \end{aligned} \]

\[ \begin{aligned} \vec{F_s}&={F_{s,x}}\vec{i}+{F_{s,y}}\vec{j}+{F_{s,z}}\vec{k}\\ &=({p_x}dydz)\vec{i}+({p_y}dxdz)\vec{j}+({p_z}dxdy)\vec{k} \end{aligned} \]

  • 体积力\(x\)方向：流体微团\(x\)方向的体积力大小\({F_{b,x}}={f_{b,x}}\rho dxdydz\)
  • 表面力\(y\)方向：流体微团\(x\)方向左侧的正压力大小\(F_{s,x,left}=[p-\frac{\partial{p}}{\partial{x}}{\frac{dx}{2}}]dydz\)，微团右侧的正压力大小为\(=F_{s,x,right}=[p+{\frac{\partial{p}}{\partial{x}}}{\frac{dx}{2}}]dydz\)

\[ \sum{F_{x}}={F_{b,x}}+{F_{s,x}} \]

\[ \begin{aligned} \sum{F_x}&=f_{b,x}\rho dxdydz+[p-\frac{\partial{p}}{\partial{x}}{\frac{dx}{2}}]dydz-[p+\frac{\partial{p}}{\partial{x}}{\frac{dx}{2}}]dydz\\ &=f_{b,x}\rho-\frac{\partial{p}}{\partial{x}}\\& =0 \end{aligned} \]

\[ f_{b,x}=\frac{1}{\rho}\frac{\partial{p}}{\partial{x}} \]

  • 体积力\(y\)方向：流体微团\(x\)方向的体积力大小\({F_{b,y}}={f_{b,y}}\rho dxdydz\)
  • 表面力\(y\)方向：流体微团\(y\)方向前侧的正压力为\(F_{s,y,front}=[p-\frac{\partial{p}}{\partial{y}}{\frac{dy}{2}}]dxdz\)，微团后侧的正压力为\(F_{s,y,back}=[p+\frac{\partial{p}}{\partial{y}}{\frac{dy}{2}}]dxdz\)

\[ f_{b,y}=\frac{1}{\rho}\frac{\partial{p}}{\partial{y}} \]

  • 体积力\(z\)方向：流体微团\(z\)方向的体积力大小\({F_{b,z}}={f_{b,z}}\rho dxdydz\)
  • 表面力\(z\)方向：流体微团\(z\)方向下侧的正压力为\(F_{s,z,down}=[p-\frac{\partial{p}}{\partial{z}}{\frac{dz}{2}}]dxdy\)，微团上侧的正压力为\(F_{s,z,up}=[p+\frac{\partial{p}}{\partial{z}}{\frac{dz}{2}}]dxdy\)

\[ f_{b,z}=\frac{1}{\rho}\frac{\partial{p}}{\partial{z}} \]

1.1.2 不同形式
  1. 正常形式

\[ \begin{cases} f_{b,x}=\frac{1}{\rho}\frac{\partial{p}}{\partial{x}}\\ f_{b,y}=\frac{1}{\rho}\frac{\partial{p}}{\partial{y}}\\ f_{b,z}=\frac{1}{\rho}\frac{\partial{p}}{\partial{z}} \end{cases} \]

  1. 矢量形式

\[ \vec{f}=\frac{1}{\rho}\nabla{p} \]

1.2泰勒公式

泰勒公式的意义在于用一个简单的多项式函数逼近一个给定的复杂函数，具有化繁为简的功能，其一般展开式如下（在\(x_0\)处展开，可以换为任意一个数）： \[ f(x)=f(x_0)+f\prime(x_0)(x-x_0)+\frac{f\prime\prime(x_0)}{2!}{(x-x_0)}^2+\cdots+r \] 也可以写为偏导数形式： \[ f(x)=f(x_0)+\frac { \partial { p } } { \partial { x } } (x-x_0)+\frac { { \partial^2(p) } } { { \partial^2(x) } } {2!}{(x-x_0)}^2+\cdots+r \] 因流体微团中心\((x,y,z)\)处的压力为\(p\)（以流体微团中心为远点），则距离中心左侧的压力为： \[ \begin{aligned} p_{s,x,left}&=p+\frac{\partial{p}}{\partial{x}}(-\frac{dx}{2}-0)\\ &=p-\frac{\partial{p}}{\partial{x}}\frac{dx}{2} \end{aligned} \] 同理，可以计算前、后、左、右、上、下六个面的压力。

2.1流体运动的描述方法

2.2随体导数

随体导数又称物质导数(Material Derivative)。

在Euler法中，速度是指某一时刻位于某一空间点处的流体质点的速度，因此速度和加速度都有关，即： \[ \vec{V}(x,y,z,t)=u(x,y,z,t)\vec{i}+v(x,y,z,t)\vec{j}+w(x,y,z,t)\vec{k} \] 加速度\(\vec{a}\)为： \[ \begin{aligned} \vec{a}&=\frac{d\vec{V}}{dt}\\ &=\frac{du}{dt}\vec{i}+\frac{dv}{dt}\vec{j}+\frac{dw}{dt}\vec{k} \end{aligned} \] 在Euler法中，因速度是关于时间和空间位置的函数即\(a_x(x,y,z,t)\)，则\(x\)方向的加速度\(a_x\)为： \[ \begin{aligned} a_x&=\frac{du}{dt}\\ &=\frac{\partial{u}}{\partial{t}}+\frac{\partial{u}}{\partial{x}}\frac{\partial{x}}{\partial{t}}+\frac{\partial{u}}{\partial{y}}\frac{\partial{y}}{\partial{t}}+\frac{\partial{u}}{\partial{z}}\frac{\partial{z}}{\partial{t}}\\ &=\frac{\partial{u}}{\partial{t}}+u\frac{\partial{u}}{\partial{x}}+v\frac{\partial{u}}{\partial{y}}+w\frac{\partial{u}}{\partial{z}} \end{aligned} \] 同理，\(y\)方向和\(z\)方向的加速度分别为： \[ a_y=\frac{\partial{v}}{\partial{t}}+u\frac{\partial{v}}{\partial{x}}+v\frac{\partial{v}}{\partial{y}}+w\frac{\partial{v}}{\partial{x}} \]

\[ a_z=\frac{\partial{w}}{\partial{t}}+u\frac{\partial{w}}{\partial{x}}+v\frac{\partial{w}}{\partial{y}}+w\frac{\partial{w}}{\partial{x}} \]

综上，加速度为： \[ \begin{aligned} \vec{a}&=[\frac{\partial{u}}{\partial{t}}+u\frac{\partial{u}}{\partial{x}}+v\frac{\partial{v}}{\partial{y}}+w\frac{\partial{w}}{\partial{z}}]\vec{i}\\ &+[\frac{\partial{v}}{\partial{t}}+u\frac{\partial{u}}{\partial{x}}+v\frac{\partial{v}}{\partial{y}}+w\frac{\partial{w}}{\partial{x}}]\vec{j}\\ &+[\frac{\partial{w}}{\partial{t}}+u\frac{\partial{u}}{\partial{x}}+v\frac{\partial{v}}{\partial{y}}+w\frac{\partial{w}}{\partial{x}}]\vec{k} \end{aligned} \] 将其写为矢量形式： \[ \begin{aligned} \vec{a}&=\frac{\partial{D}}{\partial{t}}\\ &=\frac{\partial{\vec{V}}}{\partial{t}}+(\vec{V}\cdot\nabla){\vec{V}} \end{aligned} \]

2.3雷诺输运方程

3.基本方程

3.1质量守恒定律 Mass Equation

质量守恒定律又叫连续性方程，首先从微控制体即微分的角度进行连续性方程的推导。

假设流体由左侧、下侧和前侧流入微控制体，由右侧、上侧和后侧流出微控制体。由此可以计算流入和流出微控制体的质量，并将两者做差即可得到微控制体质量的变化。

在进行计算之前，首先了解一个概念质量流量，即单位时间内流过控制面的质量，三个流入方向上的质量流量分别为： \[ \begin{cases} x:\rho{u}\\ y:\rho{v}\\ z:\rho{w} \end{cases} \] 则单位时间内由三个方向流入控制体的质量为： \[ \begin{cases} x:\rho{u}{\rm dydz}\\ y:\rho{v}{\rm dxdz}\\ z:\rho{w}{\rm dxdy} \end{cases} \] 由泰勒展开式可知，三个流出方向上的质量流量为： \[ \begin{cases} x:\rho{u}+\frac{\partial{(\rho{u})}}{\partial{x}}{\rm dx}\\ y:\rho{v}+\frac{\partial{(\rho{v})}}{\partial{y}}{\rm dy}\\ z:\rho{w}+\frac{\partial{(\rho{w})}}{\partial{z}}{\rm dz} \end{cases} \] 则单位时间内由三个方向流出控制体的质量为： \[ \begin{cases} x:[\rho{u}+\frac{\partial{(\rho{u})}}{\partial{x}}{\rm dx}]\\ y:[\rho{v}+\frac{\partial{(\rho{v})}}{\partial{y}}{\rm dy}]\\ z:[\rho{w}+\frac{\partial{(\rho{w})}}{\partial{z}}{\rm dz}] \end{cases} \] 三个方向的质量差值可得： \[ \begin{cases} x:[\rho{u}+\frac{\partial{(\rho{u})}}{\partial{x}}{\rm dx}]{\rm {dydz}}-\rho{u}{\rm dydz}=\frac{\partial{(\rho{u})}}{\partial{x}}{\rm dxdydz}\\ y:[\rho{v}+\frac{\partial{(\rho{v})}}{\partial{y}}{\rm dy}]{\rm {dxdz}}-\rho{v}{\rm dxdz}=\frac{\partial{(\rho{v})}}{\partial{y}}{\rm dxdydz}\\ z:[\rho{w}+\frac{\partial{(\rho{w})}}{\partial{z}}{\rm dz}]{\rm {dxdy}}-\rho{w}{\rm dxdy}=\frac{\partial{(\rho{w})}}{\partial{z}}{\rm dxdydz} \end{cases} \] 从整个微控制体着手，单位时间内流体质量的增加为： \[ \begin{aligned} \frac{\partial{m}}{\partial{t}}&=\frac{\partial{\rho{B}}}{\partial{t}}\\ &=\frac{\partial{\rho}}{\partial{t}} \end{aligned} \] 由质量守恒定律可知，流体质量的增加等于流入控制体的质量： \[ -\frac{\partial{\rho{ {\rm dxdydz} }}}{\partial{t}}=\frac{\partial{(\rho{u})}}{\partial{x}}{\rm dxdydz}+\frac{\partial{(\rho{v})}}{\partial{y}}{\rm dxdydz}+\frac{\partial{(\rho{w})}}{\partial{z}}{\rm dxdydz} \] 化简后可得： \[ -\frac{\partial{\rho}}{\partial{t}}=\frac{\partial{(\rho{u})}}{\partial{x}}+\frac{\partial{(\rho{v})}}{\partial{y}}+\frac{\partial{(\rho{w})}}{\partial{z}}\\ \frac{\partial{\rho}}{\partial{t}}+\frac{\partial{(\rho{u})}}{\partial{x}}+\frac{\partial{(\rho{v})}}{\partial{y}}+\frac{\partial{(\rho{w})}}{\partial{z}}=0 \] 矢量形式为： \[ \frac{\partial{\rho}}{\partial{t}}+\nabla{\cdot}(\rho{\vec{V}})=0 \]

3.2动量守恒定律 Momentum Equation

3.2.1一般形式的动量方程和无粘流动的动量方程

由牛顿第二定律可知： \[ \begin{aligned} \vec{F}&=\frac{\rm dp}{\rm dt}\\ &=m\vec{a}\\ &=\rho{\rm dxdydz}\frac { \rm { D\vec { V } } } {\rm{Dt}} \end{aligned} \] 作用于流体微团的力可以分为体积力和表面力，体积力如下，其中\(\vec{f_b}\)为单位质量的体积力： \[ \vec{F_b}=\vec{f_b}\rho{\rm dxdydz} \] 接下来是表面力： \[ \begin{cases} f_{s,x}=\frac { \partial { \tau_ { xx } } }{\partial{x}}{\rm dx}{\rm dydz}+\frac { \partial { \tau_ { yx } } }{\partial{y}}{\rm dy}{\rm dxdz}+\frac { \partial { \tau_ { zx } } }{\partial{z}}{\rm dz}{\rm dxdy}\\ f_{s,y}=\frac { \partial { \tau_ { xy } } }{\partial{x}}{\rm dx}{\rm dydz}+\frac { \partial { \tau_ { yy } } }{\partial{y}}{\rm dy}{\rm dxdz}+\frac { \partial { \tau_ { zy } } }{\partial{z}}{\rm dz}{\rm dxdy}\\ f_{s,z}=\frac { \partial { \tau_ { xz } } }{\partial{x}}{\rm dx}{\rm dydz}+\frac { \partial { \tau_ { yz } } }{\partial{y}}{\rm dy}{\rm dxdz}+\frac { \partial { \tau_ { zz } } }{\partial{z}}{\rm dz}{\rm dxdy} \end{cases} \] 综上： \[ \vec{F}=\vec{F_b}+\vec{F_s} \] 对上式化简： \[ \begin{aligned} \frac { \rm { D\vec { V } } } {\rm{Dt}}\rho{\rm dxdydz}&=\vec{f_b}\rho{\rm dxdydz}\\ &+(\frac { \partial { \tau_ { xx } } }{\partial{x}}{\rm dx}{\rm dydz}+\frac { \partial { \tau_ { yx } } }{\partial{y}}{\rm dy}{\rm dxdz}+\frac { \partial { \tau_ { zx } } }{\partial{z}}{\rm dz}{\rm dxdy})\vec{i}\\ &+(\frac { \partial { \tau_ { xy } } }{\partial{x}}{\rm dx}{\rm dydz}+\frac { \partial { \tau_ { yy } } }{\partial{y}}{\rm dy}{\rm dxdz}+\frac { \partial { \tau_ { zy } } }{\partial{z}}{\rm dz}{\rm dxdy})\vec{j}\\ &+(\frac { \partial { \tau_ { xz } } }{\partial{x}}{\rm dx}{\rm dydz}+\frac { \partial { \tau_ { yz } } }{\partial{y}}{\rm dy}{\rm dxdz}+\frac { \partial { \tau_ { zz } } }{\partial{z}}{\rm dz}{\rm dxdy})\vec{k} \end{aligned} \] 对上式同时消去\({\rm dxdydz}\)后得： \[ \begin{aligned} \frac { \rm { D\vec { V } } } {\rm{Dt}}&=\vec{f_b}\\ &+\frac{1}{\rho}(\frac { \partial { \tau_ { xx } } }{\partial{x}}+\frac { \partial { \tau_ { yx } } }{\partial{y}}+\frac { \partial { \tau_ { zx } } }{\partial{z}})\vec{i}\\ &+\frac{1}{\rho}(\frac { \partial { \tau_ { xy } } }{\partial{x}}+\frac { \partial { \tau_ { yy } } }{\partial{y}}+\frac { \partial { \tau_ { zy } } }{\partial{z}})\vec{j}\\ &+\frac{1}{\rho}(\frac { \partial { \tau_ { xz } } }{\partial{x}}+\frac { \partial { \tau_ { yz } } }{\partial{y}}+\frac { \partial { \tau_ { zz } } }{\partial{z}})\vec{k} \end{aligned} \] 将其表示为矢量形式为，需注意\(\tau\)为二阶张量： \[ \frac { \rm { D\vec { V } } } {\rm{Dt}}=\vec{f_b}+\frac{1}{\rho}\nabla\cdot\tau \] 该公式便为动量方程Momentum Equation，但是不实用因为其中以下六个分量\(\tau_{xy},\tau_{yx},\tau_{xz},\tau_{zx},\tau_{yz},\tau_{zy}\)并不容易求得。所以，欧拉基于流体的粘性力相较于压力较小，在一些问题的求解中可以忽略不记，并给出以下公式： \[ \begin{cases} \tau_{xy}=\tau_{yx}=0\\ \tau_{xz}=\tau_{zx}=0\\ \tau_{yz}=\tau_{zy}=0\\ \tau_{xx}=\tau_{yy}=\tau_{zz}=-p \end{cases}\\ \] 将上式代入动量方程后可得： \[ \frac { \rm { D\vec { V } } } {\rm{Dt}}=\vec{f_b}-\frac{1}{\rho}\nabla{p} \] 该公式为无粘流动的动量方程，最早由欧拉提出，一般称为欧拉方程。公式左侧为单位质量流体动量随时间的改变，右侧第一项为体积力，右侧第二项为压力差。该公式的物理意义是：当流动为 无粘流动时，流体的运动只由两种力产生，一种时体积力，一种是压差力。对于体积力为重力的无粘流动，如果一个质点在加速，要么是在下落要么是在从高压区流向低压区。

对于静止的流体，其公式左侧的一项为零，则可以化简为，为Euler静力学平衡方程： \[ \vec{f_b}=\frac{1}{\rho}\nabla{p} \]

3.2.2加入剪切力具体表达形式的动量方程

牛顿内摩擦定理为： \[ \tau=\mu\frac{\partial{u}}{\partial{y}} \] 该公式只是表示作用于垂直于\(y\)轴的平面上指向\(x\)轴的剪切力，实际上为\(\tau_{yx}\)，当流动不是平行流动时，牛顿流体的剪切力的表达形式为（本构方程1）： \[ \begin{cases} \tau_{xy}=\tau{yx}=\mu(\frac{\partial{v}}{\partial{x}}+\frac{\partial{u}}{\partial{y}})\\ \tau_{xz}=\tau{zx}=\mu(\frac{\partial{w}}{\partial{x}}+\frac{\partial{u}}{\partial{z}})\\ \tau_{yz}=\tau{zy}=\mu(\frac{\partial{w}}{\partial{y}}+\frac{\partial{u}}{\partial{z}}) \end{cases} \] 在欧拉方程中，我们假设\(\tau_{xx}=\tau_{yy}=\tau_{zz}=-p\)，其实正应力中并不是只有压力，粘性力也会产生一部分正应力，否则流体微元就不能满足平衡关系了。因此，斯托克斯Stoke提出了正应力关系（本构方程2）： \[ \begin{cases} \tau_{xx}=2\mu\frac{\partial{u}}{\partial{x}}-\frac{2}{3}\mu(\nabla\cdot\vec{V})-p\\ \tau_{yy}=2\mu\frac{\partial{y}}{\partial{y}}-\frac{2}{3}\mu(\nabla\cdot\vec{V})-p\\ \tau_{zz}=2\mu\frac{\partial{w}}{\partial{z}}-\frac{2}{3}\mu(\nabla\cdot\vec{V})-p \end{cases} \] 将其化简后可得正应力的展开形式： \[ \begin{cases} \begin{aligned} \tau_{xx}&=2\mu\frac{\partial{u}}{\partial{x}}-\frac{2}{3}\mu(\nabla\cdot\vec{V})-p\\ &=2\mu\frac{\partial{u}}{\partial{x}}-\frac{2}{3}\mu\frac{\partial{u}}{\partial{x}}-\frac{2}{3}\mu\frac{\partial{v}}{\partial{y}}-\frac{2}{3}\mu\frac{\partial{w}}{\partial{z}}-p\\ &=\frac{4}{3}\mu\frac{\partial{u}}{\partial{x}}-\frac{2}{3}\mu\frac{\partial{v}}{\partial{y}}-\frac{2}{3}\mu\frac{\partial{w}}{\partial{z}}-p\\ \end{aligned}\\ \begin{aligned} \tau_{yy}&=2\mu\frac{\partial{v}}{\partial{y}}-\frac{2}{3}\mu(\nabla\cdot\vec{V})-p\\ &=2\mu\frac{\partial{v}}{\partial{y}}-\frac{2}{3}\mu\frac{\partial{u}}{\partial{x}}-\frac{2}{3}\mu\frac{\partial{v}}{\partial{y}}-\frac{2}{3}\mu\frac{\partial{w}}{\partial{z}}-p\\ &=\frac{2}{3}\mu\frac{\partial{u}}{\partial{x}}-\frac{4}{3}\mu\frac{\partial{v}}{\partial{y}}-\frac{2}{3}\mu\frac{\partial{w}}{\partial{z}}-p\\ \end{aligned}\\ \begin{aligned} \tau_{zz}&=2\mu\frac{\partial{w}}{\partial{z}}-\frac{2}{3}\mu(\nabla\cdot\vec{V})-p\\ &=2\mu\frac{\partial{w}}{\partial{z}}-\frac{2}{3}\mu\frac{\partial{u}}{\partial{x}}-\frac{2}{3}\mu\frac{\partial{v}}{\partial{y}}-\frac{2}{3}\mu\frac{\partial{w}}{\partial{z}}-p\\ &=\frac{2}{3}\mu\frac{\partial{u}}{\partial{x}}-\frac{2}{3}\mu\frac{\partial{v}}{\partial{y}}-\frac{4}{3}\mu\frac{\partial{w}}{\partial{z}}-p \end{aligned} \end{cases} \] 其中，粘性正应力为： \[ \begin{cases} \begin{aligned} \tau_{xx}&=2\mu\frac{\partial{u}}{\partial{x}}-\frac{2}{3}\mu(\nabla\cdot\vec{V})\\ &=\frac{4}{3}\mu\frac{\partial{u}}{\partial{x}}-\frac{2}{3}\mu\frac{\partial{v}}{\partial{y}}-\frac{2}{3}\mu\frac{\partial{w}}{\partial{z}}\\ \end{aligned}\\ \begin{aligned} \tau_{yy}&=2\mu\frac{\partial{v}}{\partial{y}}-\frac{2}{3}\mu(\nabla\cdot\vec{V})\\ &=\frac{2}{3}\mu\frac{\partial{u}}{\partial{x}}-\frac{4}{3}\mu\frac{\partial{v}}{\partial{y}}-\frac{2}{3}\mu\frac{\partial{w}}{\partial{z}}\\ \end{aligned}\\ \begin{aligned} \tau_{zz}&=2\mu\frac{\partial{w}}{\partial{z}}-\frac{2}{3}\mu(\nabla\cdot\vec{V})\\ &=\frac{2}{3}\mu\frac{\partial{u}}{\partial{x}}-\frac{2}{3}\mu\frac{\partial{v}}{\partial{y}}-\frac{4}{3}\mu\frac{\partial{w}}{\partial{z}} \end{aligned} \end{cases} \] 针对不可压缩流体\(\nabla\cdot\vec{V}=0\)，可得粘性正应力为： \[ \begin{cases} \tau_{xx}&=2\mu\frac{\partial{u}}{\partial{x}}\\ \tau_{yy}&=2\mu\frac{\partial{v}}{\partial{y}}\\ \tau_{zz}&=2\mu\frac{\partial{w}}{\partial{z}} \end{cases} \] 由上式可见，粘性正应力一般为正直即表示为拉力，其相对于压力较小，可忽略不计。

将两个本构方程带入动量方程可得著名的纳维-斯托克斯方程，简称N-S方程，其展开形式为： \[ \begin{cases} \begin{aligned} \rho\frac{\partial{u}}{\partial{t}}+\rho{u}\frac{\partial{u}}{\partial{x}}+\rho{v}\frac{\partial{u}}{\partial{y}}+\rho{w}\frac{\partial{u}}{\partial{z}}&=\rho{f_{b,x}}\\ &+2\mu\frac{\partial{\frac{\partial{u}}{\partial{x}}}}{\partial{x}}-\frac{2}{3}\mu\frac{\partial({\frac{\partial{u}}{\partial{x}}+\frac{\partial{v}}{\partial{y}}+\frac{\partial{w}}{\partial{z}})}}{\partial{x}}-\frac{\partial{p}}{\partial{x}}\\ &+\frac{\mu(\frac{\partial{u}}{\partial{y}}+\frac{\partial{v}}{\partial{x}})}{\partial{y}}\\ &+\frac{\mu(\frac{\partial{u}}{\partial{z}}+\frac{\partial{w}}{\partial{x}})}{\partial{z}} \end{aligned}\\ \begin{aligned} \rho\frac{\partial{v}}{\partial{t}}+\rho{u}\frac{\partial{v}}{\partial{x}}+\rho{v}\frac{\partial{v}}{\partial{y}}+\rho{w}\frac{\partial{v}}{\partial{z}}&=\rho{f_{b,y}}\\ &+\mu\frac{\partial{(\frac{\partial{v}}{\partial{x}}+\frac{\partial{u}}{\partial{y}})}}{\partial{x}}\\ &+2\mu\frac{\partial{\frac{\partial{v}}{\partial{y}}}}{\partial{y}}-\frac{2}{3}\mu\frac{\partial{(\frac{\partial{u}}{\partial{x}}+\frac{\partial{v}}{\partial{y}}+\frac{\partial{w}}{\partial{z}})}}{\partial{y}}-\frac{\partial{p}}{\partial{x}}\\ &+\frac{\mu(\frac{\partial{v}}{\partial{z}}+\frac{\partial{w}}{y})}{\partial{z}}\\ \end{aligned}\\ \begin{aligned} \rho\frac{\partial{w}}{\partial{t}}+\rho{u}\frac{\partial{w}}{\partial{x}}+\rho{v}\frac{\partial{w}}{\partial{y}}+\rho{w}\frac{\partial{w}}{\partial{z}}&=\rho{f_{b,z}}\\ &+\mu\frac{\partial{(\frac{\partial{w}}{\partial{x}}+\frac{\partial{u}}{\partial{z}})}}{\partial{x}}\\ &+\mu\frac{\partial{(\frac{\partial{w}}{\partial{y}}+\frac{\partial{v}}{\partial{z}})}}{\partial{y}}\\ &+2\mu\frac{\partial{\frac{\partial{w}}{\partial{z}}}}{\partial{z}}-\frac{2}{3}\mu\frac{\partial{(\frac{\partial{u}}{\partial{x}}+\frac{\partial{v}}{\partial{y}}+\frac{\partial{w}}{\partial{z}})}}{\partial{z}}-\frac{\partial{p}}{\partial{z}} \end{aligned} \end{cases} \] 将上式进一步展开： \[ \begin{cases} \begin{aligned} \rho\frac{\partial{u}}{\partial{t}}+\rho{u}\frac{\partial{u}}{\partial{x}}+\rho{v}\frac{\partial{u}}{\partial{y}}+\rho{w}\frac{\partial{u}}{\partial{z}}&=\rho{f_{b,x}}-\frac{\partial{p}}{\partial{x}}\\ &+\mu\frac{\partial{\frac{\partial{u}}{\partial{x}}}}{\partial{x}}+\mu\frac{\partial{\frac{\partial{u}}{\partial{x}}}}{\partial{x}}-\frac{2}{3}\mu\frac{\partial({\frac{\partial{u}}{\partial{x}}+\frac{\partial{v}}{\partial{y}}+\frac{\partial{w}}{\partial{z}})}}{\partial{x}}\\ &+\mu\frac{\partial{\frac{\partial{u}}{\partial{y}}}}{\partial{y}}+\mu\frac{\partial{\frac{\partial{v}}{\partial{x}}}}{\partial{y}}\\ &+\mu\frac{\partial{\frac{\partial{w}}{\partial{x}}}}{\partial{z}}+\mu\frac{\partial{\frac{\partial{u}}{\partial{z}}}}{\partial{z}} \end{aligned}\\ \begin{aligned} \rho\frac{\partial{v}}{\partial{t}}+\rho{u}\frac{\partial{v}}{\partial{x}}+\rho{v}\frac{\partial{v}}{\partial{y}}+\rho{w}\frac{\partial{v}}{\partial{z}}&=\rho{f_{b,y}}-\frac{\partial{p}}{\partial{y}}\\ &+\mu\frac{\partial{\frac{\partial{u}}{\partial{y}}}}{\partial{x}}+\mu\frac{\partial{\frac{\partial{v}}{\partial{x}}}}{\partial{x}}\\ &+\mu\frac{\partial{\frac{\partial{v}}{\partial{y}}}}{\partial{y}}+\mu\frac{\partial{\frac{\partial{v}}{\partial{y}}}}{\partial{y}}-\frac{2}{3}\mu\frac{\partial({\frac{\partial{u}}{\partial{x}}+\frac{\partial{v}}{\partial{y}}+\frac{\partial{w}}{\partial{z}})}}{\partial{y}}\\ &+\mu\frac{\partial{\frac{\partial{w}}{\partial{y}}}}{\partial{z}}+\mu\frac{\partial{\frac{\partial{v}}{\partial{z}}}}{\partial{z}} \end{aligned}\\ \begin{aligned} \rho\frac{\partial{w}}{\partial{t}}+\rho{u}\frac{\partial{w}}{\partial{x}}+\rho{v}\frac{\partial{w}}{\partial{y}}+\rho{w}\frac{\partial{w}}{\partial{z}}&=\rho{f_{b,z}}-\frac{\partial{p}}{\partial{z}}\\ &+\mu\frac{\partial{\frac{\partial{w}}{\partial{x}}}}{\partial{x}}+\mu\frac{\partial{\frac{\partial{u}}{\partial{z}}}}{\partial{x}}\\ &+\mu\frac{\partial{\frac{\partial{v}}{\partial{z}}}}{\partial{y}}+\mu\frac{\partial{\frac{\partial{w}}{\partial{y}}}}{\partial{y}}\\ &+\mu\frac{\partial{\frac{\partial{w}}{\partial{z}}}}{\partial{z}}+\mu\frac{\partial{\frac{\partial{w}}{\partial{z}}}}{\partial{z}}-\frac{2}{3}\mu\frac{\partial({\frac{\partial{u}}{\partial{x}}+\frac{\partial{v}}{\partial{y}}+\frac{\partial{w}}{\partial{z}})}}{\partial{y}} \end{aligned} \end{cases} \] 其中： \[ \begin{cases} \mu[\frac{\partial{\frac{\partial{u}}{\partial{x}}}}{\partial{x}}+\frac{\partial{\frac{\partial{u}}{\partial{y}}}}{\partial{y}}+\frac{\partial{\frac{\partial{u}}{\partial{z}}}}{\partial{z}}]\\ \mu[\frac{\partial{\frac{\partial{v}}{\partial{x}}}}{\partial{x}}+\frac{\partial{\frac{\partial{v}}{\partial{y}}}}{\partial{y}}+\frac{\partial{\frac{\partial{v}}{\partial{z}}}}{\partial{z}}]\\ \mu[\frac{\partial{\frac{\partial{w}}{\partial{x}}}}{\partial{x}}+\frac{\partial{\frac{\partial{w}}{\partial{y}}}}{\partial{y}}+\frac{\partial{\frac{\partial{w}}{\partial{z}}}}{\partial{z}}] \end{cases}\\ =\mu\nabla\cdot\nabla\vec{V} \] 又因： \[ \begin{cases} \mu\frac{\partial{\frac{\partial{v}}{\partial{x}}}}{\partial{y}}=\mu\frac{\partial{\frac{\partial{v}}{\partial{y}}}}{\partial{x}};\mu\frac{\partial{\frac{\partial{w}}{\partial{x}}}}{\partial{z}}=\mu\frac{\partial{\frac{\partial{w}}{\partial{z}}}}{\partial{x}}\\ \mu\frac{\partial{\frac{\partial{u}}{\partial{y}}}}{\partial{x}}=\mu\frac{\partial{\frac{\partial{u}}{\partial{x}}}}{\partial{y}};\mu\frac{\partial{\frac{\partial{w}}{\partial{y}}}}{\partial{x}}=\mu\frac{\partial{\frac{\partial{w}}{\partial{x}}}}{\partial{y}}\\ \mu\frac{\partial{\frac{\partial{u}}{\partial{z}}}}{\partial{x}}=\mu\frac{\partial{\frac{\partial{u}}{\partial{x}}}}{\partial{z}};\mu\frac{\partial{\frac{\partial{v}}{\partial{z}}}}{\partial{y}}=\mu\frac{\partial{\frac{\partial{v}}{\partial{y}}}}{\partial{z}} \end{cases} \] 因此： \[ \begin{cases} \mu[\frac{\partial{(\frac{\partial{u}}{\partial{x}}+\frac{\partial{v}}{\partial{y}}+\frac{\partial{w}}{\partial{z}})}}{\partial{x}}]\\ \mu[\frac{\partial{(\frac{\partial{u}}{\partial{x}}+\frac{\partial{v}}{\partial{y}}+\frac{\partial{w}}{\partial{z}})}}{\partial{y}}]\\ \mu[\frac{\partial{(\frac{\partial{u}}{\partial{x}}+\frac{\partial{v}}{\partial{y}}+\frac{\partial{w}}{\partial{z}})}}{\partial{z}}]\\ \end{cases} =\mu\nabla(\nabla\cdot\vec{V}) \] 综上，N-S方程的矢量形式为： \[ \frac { { \rm D } \vec { V } } { { \rm D } t } =\vec{f_b}-\frac{1}{\rho}\nabla{p}+\frac{\mu}{\rho}\nabla\cdot\nabla\vec{V}+\frac{1}{3}\frac{\mu}{\rho}\nabla(\nabla\cdot\vec{V}) \]

3.3能量守恒定律 Energy Equation

4.需要记忆的公式

4.1 欧拉静力学方程

\[ \begin{cases} f_{b,x}=\frac{1}{\rho}\frac{\partial{p_x}}{\partial{x}}\\ f_{b,y}=\frac{1}{\rho}\frac{\partial{p_y}}{\partial{y}}\\ f_{b,z}=\frac{1}{\rho}\frac{\partial{p_z}}{\partial{z}} \end{cases} \]

\[ \vec{f}=\frac{1}{\rho}\nabla{p} \]

4.2随体导数

\[ \begin{cases} \frac{du}{dt}=\frac{\partial{u}}{\partial{t}}+u\frac{\partial{u}}{\partial{x}}+u\frac{\partial{v}}{\partial{y}}+u\frac{\partial{w}}{\partial{z}}\\ \frac{dv}{dt}=\frac{\partial{v}}{\partial{t}}+v\frac{\partial{u}}{\partial{x}}+v\frac{\partial{v}}{\partial{y}}+v\frac{\partial{w}}{\partial{z}}\\ \frac{dw}{dt}=\frac{\partial{u}}{\partial{t}}+w\frac{\partial{u}}{\partial{x}}+w\frac{\partial{v}}{\partial{y}}+w\frac{\partial{w}}{\partial{z}} \end{cases} \]

\[ \frac{D\vec{V}}{Dt}=\frac{\partial{\vec{V}}}{\partial{t}}+(\nabla{\cdot}\vec{V})\vec{V} \]

\[ \frac{D\Phi}{Dt}=\frac{\partial{\Phi}}{\partial{t}}+(\nabla{\cdot}\Phi)\vec{V} \]

4.3高斯公式

高斯公式又称散度定理，可以写为： \[ \iiint({\nabla{\cdot}}\vec{\Phi}){\rm dB}=\oint({\vec{\Phi}\cdot{\vec{n}}}){\rm dS} \] 其中\(\Phi\)为任意向量，公式左侧项为任意矢量的散度在一个空间内的体积分，表示了该空间内某种量的净生成量。公式右侧项为该矢量在包围这个空间的封闭表面的通过量。该公式描述了物理性质的守恒的特性，可以理解为：一个空间内的净生成量等于净流出量。

  • 公式右侧项表示控制体(Control Volume, CV)，右侧项为包围控制体的控制面(Control Surface, CS)。

  • 该公式的主要作用为将面积分转变为体积分，使得方便化简公式。

将替\(\Phi\)换为\(\vec{V}\)后为： \[ \iiint({\nabla{\cdot}}\vec{V}){\rm dB}=\oint({\vec{V}\cdot{\vec{n}}}){\rm dS} \] 该式子表示：控制体内流体体积的增加量等于通过控制面流出的流体体积。

4.4散度的意义

由高斯公式可以推导出速度的散度的意义： \[ \nabla{\cdot}{\vec{V}}=\frac{1}{\delta{B}}\frac{ { d {(\delta { B } ) } } }{d{t}} \] 推导过程如下：

  1. 单位时间内流体控制体体积的变化为：

\[ \frac { { \rm d { B } } } { { \rm d { t } } } =\oint(\vec{V}\cdot{\vec{n}}){\rm dS} \]

  1. 因高斯公式，可有如下公式：

\[ \begin{aligned} \frac { { \rm d { B } } } { {\rm d { t } } } &=\oint(\vec{V}\cdot{\vec{n}}){\rm dS}\\ &=\iiint( { \nabla { \cdot} {\vec { V } } } ){\rm dB} \end{aligned} \]

  1. 当控制体足够小时，右侧三重积分可去除，则公式可变为：

\[ \frac { { \rm d( { \delta { B } } ) } } { {\rm dt } }=\nabla{\cdot}\vec{V}{\rm d({\delta{B})}}\\ \frac{1} { { \rm d( { \delta { B } ) } } } \frac{ { \rm d( { \delta { B } } ) } } { { \rm dt } } =\nabla{\cdot}\vec{V} \]

4.5质量守恒方程Conservation Equation

质量守恒方程又称连续性方程，以下方程是从Euler法的角度进行推导的结果： \[ \frac{\partial\rho}{\partial{t}}+\nabla\cdot(\rho\vec{V})=0 \] 展开式如下： \[ \frac{\partial\rho}{\partial{t}}+\frac{ \partial{ \rho{ u } } }{\partial{x}}+\frac{ \partial{ \rho{ v } } }{\partial{y}}+\frac{ \partial{ \rho{ w } } }{\partial{z}}=0 \]
